Security Policy

Supported Versions

These are the version ranges of HydePHP, and their support status. We follow Semantic Versioning, and you can read about our Backwards Compatability promise here.

Version Supported Classification
1.x LTS :white_check_mark: General Availability
< 0.64 :x: Beta
< 0.8 :x: Alpha

Reporting a Vulnerability

If you discover a security vulnerability within this package, please send an e-mail to the creator, Caen De Silva, via caen@desilva.se. You can also report a vulnerability through GitHub on the Security Advisory page.

All security vulnerabilities will be promptly addressed.
